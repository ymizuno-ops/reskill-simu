================================================================================
  リスキリング年収シミュレーター
  Reskilling Income Simulator
================================================================================

概要
----
厚生労働省「賃金構造基本統計調査」（2006〜2024年）・GDP統計・消費者物価指数を
もとに機械学習モデルを構築し、リスキリング・転職を行った場合の
50年間の年収推移をシミュレーションするWebアプリケーションです。

「平均年収」を見るだけの既存ツールとは異なり、
  ・現在の年齢・勤続年数・実績年収
  ・転職先職種・スキル引継ぎ率
  ・GDP成長率・物価指数・リアリティ補正
を組み合わせた、パーソナライズされた収支分析が可能です。

セットアップ手順
--------------------------------------------------------------------------------

【1】依存ライブラリのインストール

    pip install -r requirements.txt

    ※ LightGBM / CatBoost / XGBoost は以下でインストール
    pip install lightgbm catboost xgboost


【2】e-stat からデータを取得

    以下の URL からデータをダウンロードし、対応するサブディレクトリに配置します。
    https://www.e-stat.go.jp/

    データ名                              配置先サブディレクトリ
    ──────────────────────────────────────────────────────────────
    職種別きまって支給する現金給与額      職種別きまって支給する現金給与額/
    年齢階級別きまって支給する現金給与額  年齢階級別きまって支給する現金給与額/
    経験年数階級別きまって支給する        経験年数階級別きまって支給する現金給与額/
      現金給与額
    国民経済計算（GDP統計）               国民経済計算_GDP統計/
    消費者物価指数                        消費者物価指数/
    毎月勤労統計調査 結果確報             毎月勤労統計調査_結果確報/

    ※ .xls 形式のファイルは LibreOffice で .xlsx に変換してから配置してください。
       LibreOffice がインストールされている場合は自動変換も可能です。


【3】データ処理・モデル訓練（初回のみ）

    # Step1: raw データを processed CSV に変換
    python src/step1_to_processed.py

    # Step2: ML 用マスターデータを構築
    python src/step2_to_master.py

    # Step3: 全モデルを訓練・保存（Stacking を含む場合は 10〜20 分程度）
    python src/step3_train.py


【4】アプリ起動

    streamlit run app.py

    ブラウザで http://localhost:8501 を開いてください。

--------------------------------------------------------------------------------
シミュレーションロジック
--------------------------------------------------------------------------------

【転職初年度年収の算出】

  ベース年収  = 転職先職種の「現在の年齢より 1 段階下の年齢階級」の平均年収
              （例: 30 歳転職 → 25〜29 歳階級の平均が基準）

  初年度年収  = ベース年収
              + (同年齢・同勤続の経験者年収 - ベース年収) × スキル引継ぎ率

  スキル引継ぎ率の目安:
     0% ……… 完全未経験スタート（1 段下の年齢階級相当）
    20% ……… 前職の汎用スキルが少し評価される
    40% ……… ドメイン知識がある程度活かせる
    60% ……… 近接領域・類似業務からの転職
    80% ……… かなりのスキルが転用できる
   100% ……… 即戦力（資格・経験が完全移行）

【現状維持シナリオ】

  AI モデルの予測値に「実績年収との乖離補正」を適用し、
  マクロ賃金上昇率（GDP 成長率 + CPI）を毎年加算します。

【リアリティ補正パラメータ】

  転職後の昇給抑制 ……… 転職後序盤（〜10 年）の昇給を抑制（0〜50%）
  キャリアリスク係数 …… 10 年以降から蓄積する技術陳腐化・頭打ちリスク（0〜30%）

--------------------------------------------------------------------------------
注意事項
--------------------------------------------------------------------------------

・本シミュレーターは統計的推計に基づくものであり、
  個人の実際の収入を保証するものではありません。

・転職市場の需給・景気変動・個人のパフォーマンス等は考慮されていません。

・データは定期的に最新年のものに更新することを推奨します。
  （毎年 e-stat に新しいデータが公開されます）

・Stacking Ensemble モデルの訓練には 10〜20 分程度かかります。
  訓練済みの models.pkl がある場合は再訓練不要です。
